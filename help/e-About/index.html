Contact

    If you have any suggestion or question about the use of MaxFEM, you can send it by email.

    If you have problems with MaxFEM, before contact us please see the Software requirements, Installation and Known issues sections of User guide in the help documentation of MaxFEM. Be aware that you can consult those sections even when the MaxFEM window is not available in the HTML file INSTALLDIR/help/b-User guide/Introduction.html (where INSTALLDIR is the MaxFEM installation folder).

    If the problem persists, please contact us indicating:

    • Name and version of your operating system
    • Name and version of your Python distribution
