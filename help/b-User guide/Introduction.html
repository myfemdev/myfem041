1.1. What is MaxFEM?

MaxFEM is a free software package for numerical simulation in electromagnetism covering electrostatics, direct current, magnetostatics and eddy currents. In addition to the code itself, MaxFEM also includes a user interface to facilitate its handling. Especially important is the reconfigurable nature of it, which easily adapts to the numerical solution of other physical and mathematical problems.

MaxFEM is fitted with an effective help manual. The manual consists of a set of independent sections where the different modules composing the package are described. In particular, the equations modeling each electromagnetic phenomenon are introduced and some examples illustrating different applications are explained.

This structure reflects the modular architecture of the software where new features can be implemented without any changes in the main program connecting all the modules.

1.7. Typographical convention

The following typographical conventions will be used throughout this help manual:

  • A bold font indicates that the given word(s) appear exactly in that way on the graphical user interface. For instance, we often refer to Project, which is the menu that appears in the left upper part of the graphical interface.

  • An arrow → indicates a sequence of steps in the menu items to achieve a particular task. For example, the sequence Project → Applications → Electrostatics 2D is equivalent to select the Project menu, display the Application submenu and choose the item label Electrostatic 2D.

  • An italic bold font refers to labels and buttons in the dialog boxes appearing on the graphical user interface. For example, the sequence Visualization → Mesh → Choose representation, implies that when you display the Visualization menu and select the Mesh submenu, the label Choose representation indicates that the user must choose one on the options given below for the representation of the mesh.

  • An italic font indicates keyboard entries in the user interface.

  • A blue font indicates a link.
