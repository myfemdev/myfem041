3.1. Electrostatics 2D

□ What MaxFEM Solves

The MaxFEM Electrostatics 2D application numerically solves electrostatics bidimensional problems in Cartesian coordinates. You can find extensive information about the electrostatics mathematical model in the MaxFEM Models user guide.

MaxFEM allows to consider linear dielectric materials or conducting media as well as different kinds of charge densities. The admissible options both in the 2D and 3D cases are summarized in the tables below.

Material properties (ε) 2D 3D
Linear and isotropic Yes Yes
Linear and orthotropic Yes Yes

Admisible charges Cartesian 2D Cartesian 3D
Point charges -- Yes
Linear charges Yes Yes
Surface charges Yes Yes
Volumetric charges Yes Yes

It is important to note that the nomenclature for the 2D case comes from the source applied in the 3D case. For instance, the term “line charge” in 2D refers to a point source in 2D which comes from a line charge in 3D.

For the numerical approximation of electrostatics problems, MaxFEM uses Lagrange finite elements of order one to compute the electric potential V. Thus the electric field is approximated by piecewise functions in each element of the mesh (triangles in 2D and tetrahedra in 3D).

To use the electrostatic application the user needs:

  1. To provide a finite element mesh of the domain: menu Mesh.

  2. To impose the boundary conditions on the boundaries: menu Boundary Conditions.

  3. To assign materials to the different components of the domain: menu Properties. Materials must be previously selected and/or defined in the Materials database.

  4. To assign source properties: menu Sources.

  5. To solve the problem in direct or remote mode: menu Solver/Run or Solver/Run remote, respectively. The user can stop the computation before it is completed in the label Solver/Stop.

  6. Once the problem has been solved, various quantities and fields based on the potential can be computed. It is also possible to create displays of the results: menu Visualization.

All the previous steps are described in the items below.

□ The Boundary Conditions

Boundary conditions is a menu entry to define boundary or interface conditions and attach them to boundaries. Electrostatics application considers two types of boundary conditions.

Dirichlet boundary conditions

In this case, the value VD of the electrostatic potential on boundary ∂ΩD must be given. To do that, you need:

  1. Define the boundary condition (for instance, blocking_1)

    Boundary conditions → Dirichlet: electrostatic potential, V (V) → blocking_1 → Add

  2. Define the boundary condition via a constant or a precompiled function

    Dirichlet conditions: blocking_1

    The condition is given by: A constant or a function

  3. Assign the boundary condition to the corresponding boundary reference(s). For instance:

    The condition is given by → A constant → Line references → 1 2 4

  4. If VD takes a constant value, define this value:

    The condition is given by: A constant → Constant value → 0.10

  5. If VD is a precompiled function, select “Function defined by user”:

    The condition is given by: A function → Function name → Function defined by user

  6. Then, you must recompilate the solver before running it.

□ The Sources

In the electrostatic 2D application, sources can be:

  • Line charge density ρl (C/m)
  • Surface charge density, ρs (C/m2)
  • Volumetric charge density, ρv (C/m3)

To impose a charge density, you must perform the following steps depending on the type of charge.

Line charge density

  1. Choose the type of charge density in the Sources item of the application.

    Sources → Line charge density, ρl (C/m)

  2. Then, define a new charge and press the button Add of the dialog box.

    Line sources: source _1 → Add

  3. Press the name source _1 you have just created. Then a new dialog box appears where you must introduce the coordinates of the point(s) where the charge is imposed and its value(s). Notice that you can introduce an array of points where you can define the corresponding charge value.

Surface charge density

  1. Choose the type of charge density in the Sources item of the application.

    Sources → Surface charge density, ρs (C/m2)

  2. Then, define a new charge and press the button Add of the dialog box.

    Line sources: source _1 → Add

  3. Press the name source _1 you have just created. Then a new dialog box appears where you must introduce the references of the line(s) where the charge is imposed and its value. This value can be a constant or a function.

Volumetric charge density

  1. Choose the type of charge density in the Sources item of the application.

    Sources → Volumetric charge density, ρv (C/m3)

  2. Then, define a new charge and press the button Add of the dialog box.

    Volumetric sources: source _1 → Add

  3. Press the name source _1 you have just created. Then a new dialog box appears where you must introduce the references of the surfaces(s) where the charge is imposed and its value. This value can be a constant or a function.

    Example 4: Potential

    Statement of the problem

    In this example two elliptical infinite hollow cylinders have potential of opposite sign, +1 and -1, respectively. The electric potential distribution due to this potential drop will be computed.

    The electric field and the electric potential are the same in any plane transversal to the axis of the cylinders. Then, as computational domain we consider the XY plane. For the numerical resolution, we consider an artificial boundary, a circumference of center (0,0) and radius 5, enclosing the two ellipses and place far enough to consider that the potential in this boundary is equal to 0.

     

    Resolution procedure

    1. Once MaxFEM is open, the user must select the application Electrostatic2D from the Project of the menu bar.

      Project → Applications→ Electrostatic2D

      Remember that before start to work with a particular application, the user must choose a working directory.

      Then, from the item Project of the Menu bar, select

      Project → Sample Data→ Example 4: Potential

    2. The computational domain is a circle of center (0,0) and radius 2. The user must upload the mesh malla_ej4.mfm from the Mesh item of the menu bar.

      Mesh → open → Choose mesh file → malla_ej4.mfm

      The user can display the mesh of the computational domain by choosing the option Visualization → Mesh of the Menu bar. If not, the computational domain will be displayed when selecting any item from the menu bar.

      After we have the mesh, we start to go through the menu bar from the left to right and we choose the different options related to the whole simulation.

    3. To assign the different materials to the computational domain, the user must:

      1. Select the item Properties → Materials of the Menu bar.

      2. In the Choose surface references dialog box, select the reference 1.

      3. The dialog box Choose material appears; by default, the material named Test material 1 associated to this reference domain is displayed. If not, the user must choose it. The domain is colored in blue in the visualization window.

      The user can display the different properties of this material in Materials database item of the menu bar.

       

    4. Now we are going to impose the boundary conditions. Three different Dirichlet boundary conditions are considered: a homogenous boundary condition in the outer boundary; potential V=-1 on the left ellipse and V=1 on the right one. To impose this boundary conditions the user must:

      1. Select the option Boundary conditions → Dirichlet: electric scalar potential in the menu bar.

      2. Click on Condition 1 on the Dirichlet conditions dialog box.

      3. In the dialog box The condition is given by, select the option A constant

      4. Click on Line reference. By default, number 1 2 7 and 8 associated to the different parts of this boundary must be stand out in the in the box below. If not, click on each number to identify each reference in the visualization window and select the appropriate ones.

      5. Click on Constant value and type 0 in the box below.

      6. Click on Condition 2 on the Dirichlet conditions dialog box.

      7. In the dialog box The condition is given by, select the option A constant

      8. Click on Line reference. By default, number 3 4 5 and 6 associated to the different parts of this boundary must be stand out in the in the box below. If not, click on each number to identify each reference in the visualization window and select the appropriate ones.

      9. Click on Constant value and type 1 in the box below.

      10. Click on Condition 3 on the Dirichlet conditions dialog box.

      11. In the dialog box The condition is given by, select the option A constant

      12. Click on Line reference. By default, number 9 10 11 and 12 associated to the different parts of this boundary must be stand out in the in the box below. If not, click on each number to identify each reference in the visualization window and select the appropriate ones.

      13. Click on Constant value and type -1 in the box below.

      14. Click on the item Sources of the Menu bar to continue with the resolution procedure.

    5. In this example no sources have been considered.

    6. Select the item Run (or run remote depending on your preference) of the menu bar to proceed with the resolution of the problem.

    7. Select the item Visualization of the menu bar to display the results. For instance,

      1. Select Mesh → Triangulation to display the mesh of the computational domain.

         

        Figure 1: Mesh of the computational domain.

      2. Choose the option Plot over line to display both the isosurfaces and the potential along a cut plane with the 3D domain. Select iso view to display the line along with the potential is represented. You can move the first and second points defining this line by selecting them with the mouse.

        Figure 2: Plot over line representation of the potential.

      3. Choose the option 3D Plot , Scale in the Z axis 3 to display the figure below.

        Figure 3: 3D representation of the potential V.

      4. Choose the option Visualization → Electric field of the menu bar. Select the option Vector components and then Modulus in the box below to display the modulus of the electric field.

         

        Figure 6: Modulus of the vector field E.

      5. Select the option Vector to display the vector electric field.

         

        Figure 7: Electric field E (scale 0.5, density 100%).
