3.9. Eddy currents 3D

□ What MaxFEM Solves

The MaxFEM Eddy Currents 3D application numerically solves eddy currents three-dimensional problems in Cartesian coordinates. You can find extensive information about the eddy currents mathematical model in the MaxFEM Models user guide.

MaxFEM only considers linear and isotropic materials in this application. For the numerical approximation in 3D, MaxFEM uses a mixed formulation with different finite elements (see eddy currents mathematical model in the MaxFEM Models user guide for a complete description of the discretization).

To use the eddy currents application the user needs:

  1. To provide a finite element mesh of the domain: menu Mesh.

  2. To impose the boundary conditions on the boundaries: menu Boundary Conditions.

  3. To assign materials to the different components of the domain: menu Properties. Materials must be previously selected and/or defined in the Materials database.

  4. To assign the physical parameters: menu Data.

  5. To solve the problem in direct or remote mode: menu Solver/Run or Solver/Run remote, respectively. The user can stop the computation before it is completed in the label Solver/Stop.

  6. Once the problem has been solved, various quantities and fields based on the potential can be computed. It is also possible to create displays of the results: menu Visualization.

All the previous steps are described in the items below.

□ The Mesh

MaxFEM does not include a mesh generator tool so the user must provide its own meshes. In 3D eddy current application, mesh files only can be in mfm format (see section MaxFEM mesh files for a complete description of the mesh formats). Morever, two tetrahedral meshes must be provided: a mesh for the whole domain, where the nodes must be the edges of the tetrahedra, and other mesh for the dielectric subdomain with nodes in the vertex. The meshes may be loaded from the Mesh menu by choosing

Mesh → Domain mesh → Choose mesh file → meshfile
Mesh → Dielectric mesh → Choose mesh file → meshfile

□ The Boundary Conditions

Boundary conditions is a menu entry to define boundary or interface conditions and attach them to boundaries. In Eddy currents 3D application the sources of the problem are introduced through boundary conditions by providing the current intensity on the conducting parts which are directly connected to an external circuit. Thus, this application considers two types of boundary conditions: Source current intensity condition on the surface of conductors where the current enters (or leaves) the domain and Dielectric boundary conditions imposed on the boundary of the dielectric domain. For this latter, the user has to provide only the connected components of the dielectric boundary with the restriction that the first connected component must be that which includes the interface of the conductors connected to the electric current (see mathematical model).

Dielectric boundary

In this case, the user must specify each connected component of the boundary of the dielectric domain. The first connected component must be the one which touch the inductors parts. To do this, you need:

  1. Define the boundary condition (for instance, Condition 1)

    Boundary conditions → Dielectric boundary → Condition 1 → Add

  2. Choose the surface reference(s). For instance:

    Choose surface references → 2, 3, 7

Source current intensity

In this case, the value I of the current intensity on boundary ΓJ must be given. To do that, you need:

  1. Define the boundary condition (for instance, Condition 3)

    Boundary conditions → Source current intensity, I (A) → Condition 3 → Add

  2. Assign the current intensity to the corresponding surface reference(s). For instance:

    Surface references → 7

  3. Define the constant value for the intensity and the phase angle:

    RMS value (V/m) → 1000

    Phase angle (Degrees) → 0

□ Post-Processing Quantities Available

From the resolution of the eddy currents problem, appropriate derived quantities can be computed. The post-processing quantities available in the MaxFEM eddy currents applications are summarized in the table below.

Symbol Magnitude Type Unit
J Current density V A/m2
B Magnetic flux density V T
H Magnetic field V A/m
Pd Active power density S W/m3

Eddy currents 3D application postprocessing quantities
S: Scalar quantity, V: vectorial quantity

As in the eddy currents application the post-processed vectorial quantities are complex, the user is only allowed to plot the vector fields of real parts, imaginary parts or complex modulus (except for the current density, for which only the vector fields of real and imaginary parts are available). Furthermore, for each one of these vectorial fields, it is possible to visualize the scalar fields corresponding to the x, y and z components, as well as its euclidean norm.

For instance, by choosing

Visualization → Magnetic flux density B → Real part → Vector components → Modulus

the application plots the scalar field

By choosing

Visualization → Magnetic flux density B → Modulus → Vectors

the application plots the vector field

For the current density, it is also possible to plot the modulus of the real vector corresponding to the RMS Value of the current density, which is a scalar field.

Example 1: Cylinder

Statement of the problem

In this section we present an example which has been used in [3, 5] to validate the computer code because its geometry and physical properties allow to obtain an analytical solution of the 3D eddy current model.

Figure 1 shows a sketch of the domain Ω which contains a conductor ΩC and a dielectric ΩD. ΩC and Ω are coaxial cylinders of radius RC and RD, respectively.

Figure 1: Sketch of the domain.

We consider that ΩC and Ω are bounded sections with height L of respective infinite cylinders. The electrical conductivity σ is taken as constant in ΩC and the magnetic permeability μ is constant in the whole Ω. An alternating current of intensity I(t) = I0cos(ωt) enters the conductor through ΓJ1 and goes through ΩC in the axial direction; I0 denotes the complex amplitude of the intensity and ω the angular frequency.

Example 2: Electric arc furnace. One-phase current

Statement of the problem

Figure 1: Sketch of the reduction furnace.

The objective of this example is to compute the induced currents in metallurgical electrodes used in an electric furnace devoted, for instance, for silicon production. A simple sketch of the furnace can be seen in Figure 1; it consists of a cylindrical pot containing charge materials and three electrodes disposed conforming an equilateral triangle. Electrodes are the main components of the reduction furnaces and their purpose is to conduct the electric current to the center of the furnace; the current enters the electrode through the the so called “contact clamps” and goes down crossing the column generating heat by Joule effect (see, for instance, [4]). At the tip of each electrode an electric arc is produced, generating the high temperatures that activate the chemical reaction. To simplify the geometry of the furnace, we have modeled the electrodes by imposing the current intensity which leaves the electrodes from its bottom part; thus, we have used the domain presented in Figure 2 without considering the contact clamps and its connection with the electrical network. The source current is usually a threephase alternating current and we include here two examples: Example 2 models the problem by using a one-phase current and Example 3 considers a three-phase current. The RMS value is equal to 70000 A. and the current frequency is equal to 50 Hz.

Figure 2: Domain considered in MaxFEM to model an electric furnace.

Concerning the materials, the electrodes consists of a central column made of graphite surrounded by a carbon paste (see Material Database).

Notice that in this case, the dielectric domain has only one connected component and the conducting domain has three input current zones.

Notice moreover that due to the current is normal to the input/exit current zones the current density J only has z-component and then this problem can also be solved by using the 2D eddy current model implemented in MaxFEM by working in a cross section of the pot (see Example 2 of the eddy current 2D application).

Example 4: Plate

Statement of the problem

The objective of this example is to compute the induced currents produced in a plate which is placed near a coil which carries an alternating current. This configuration appears for instance in electromagnetic forming processes; although in that case the source is a genuine transient current (not harmonic), here we will solve the 3D time-harmonic eddy current model. The coil and the plate are presented in Figure 1, which also shows a mesh of the conducting domain. The domain has been chosen as a sufficiently large box surrounding the conductor. The current intensity which enters the coil has a RMS value equal to 40000 A. and null phase, while the frequency is equal to 35000 Hz. The input current intensity is provided through one of the end points of the coil. The material of the workpiece is magnesium alloy while the coil is made of copper (see Material database).

Figure 1: Mesh of the conducting domain (left). Detail of the coil mesh (right).

Notice that in this example the boundary of the dielectric domain has two connected components.

 

Resolution procedure

  1. Once MaxFEM is open, the user must select the application Eddy Currents 3D from the Project of the menu bar.

    Project → Applications → Eddy Currents 3D

    Remember that before start to work with a particular application, the user must choose a working directory.

    Then, from the item Project of the Menu bar, select

    Project → Sample Data → Example 4: Plate

  2. The computational domain is sketched in Figure 1. The user must upload the domain mesh mallatotalplacas.mfm and the dielectric mesh malladplaca.mfm from the Mesh item of the menu bar.

    Mesh → Domain mesh → Choose mesh file → mallatotalplacas.mfm
    Mesh → Dielectric mesh → Choose mesh file → malladplaca.mfm

    The user can display the mesh of the computational domain by choosing the option Visualization → Domain/Dielectric Mesh of the Menu bar. If not, the computational domain will be displayed when selecting any item from the menu bar.

    Figure 2: Computational domain (domain mesh).

    After we have the mesh, we start to go through the menu bar from the left to right and we choose the different options related to the whole simulation.

    4. In the Choose domain references dialog box, select the reference 2.

    6. In the Choose domain references dialog box, select the reference 3.

    8. Then, assign the current intensity to the corresponding surface reference:

      RMS value (A) → 40000

      Phase angle (Degrees) → 0

  5. The item Data of the menu bar allows to introduce different data depending on the application. In eddy currents, the option allows to introduce the current frequency. In this example, the value of the frequency is 35000 Hz. To define it, the user must

    1. Select Data → Frequency in the menu bar.

    2. The dialog box Frequency (Hz) appears. By default, the value 35000 is displayed.

      Visualization → Current density → RMS Value → Filled

       

      Figure 4: Modulus of the current density in the plate.

      Visualization → Current density → Real part → Vectors
